## 📌 Project Overview

This project is part of the **BCG X Data Science Virtual Experience Program** hosted on Forage.
The objective was to analyze customer churn for PowerCo and deliver actionable, data-driven insights.

---

## 🎯 Business Problem

PowerCo is experiencing **customer churn (~10%)**.
The goal was to determine whether **price sensitivity** is the main driver or if other factors influence churn.

---

## 🧠 Approach

* Business Understanding
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Machine Learning Modeling
* Business Recommendations

## 🔹 Task 3: Feature Engineering

* Tenure & time-based features
* Encoded categorical variables
* Log transformations for skewed data
